SET Game Rules

Cards

There are 81 unique cards in total
Each card has has 4 features to it

  • Color
  • Number
  • Shape
  • Shading

Objective

The objective of the game is to make as many sets as possible

What is a set?

A set is 3 cards, such that for all features, all 3 cards must all have the same different feature
